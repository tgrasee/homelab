Infrastructure as Code

Automated Homelab
Infrastructure Stack

A fully automated home server built on Proxmox — provisioned with Terraform, configured with Ansible, monitored with Grafana, and validated on every push with a CI/CD pipeline.

See the Stack GitHub Repo
⚙️

Zero-Touch Operations

The entire server — from OS install to running services — is defined as code. A single command provisions VMs, installs software, and deploys all services. No manual steps, no configuration drift, fully repeatable.

Automated nightly backups run without intervention. If anything changes unexpectedly, it gets flagged.

🔔

Always-On Monitoring

Grafana dashboards track CPU, memory, disk, and network across every host in real time. Prometheus collects metrics every 15 seconds. Loki aggregates logs from all services.

Alerts fire to email the moment a node goes down or disk usage crosses 85% — before it becomes a problem.
